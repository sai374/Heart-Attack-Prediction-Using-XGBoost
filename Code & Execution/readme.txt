To Run this Project 

In cmd come to this file location and type pip install -r requirements.text

and then the entire supporting softwares will be installed.

The Softwares that are required to run this project are.

1.Flask
2.Pandas
3.Guicorn
4.XgBoost
5.Python 2.7
6.Sklearn

after Installation. In Cmd type python app.py
Then the localhost will be running on an port.

Copy that port and open in any browser. Then you can find all the fields that are used to predict heart attack.

In order to cross check the data. In the same folder thers is an csv file called heart. Input this values in the application
then you can see the predcition.
